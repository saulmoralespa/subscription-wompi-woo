Integración de suscripciones con Wompi para WooCommerce.

== Descripción ==
Subscription Wompi WooCommerce es un plugin que permite integrar pagos recurrentes y suscripciones en WooCommerce mediante Wompi.
Con esta herramienta, puedes ofrecer planes de suscripción a tus clientes y automatizar los cobros sin complicaciones.

### Características:
* Integración con la pasarela de pagos Wompi
* Pagos recurrentes automatizados
* Compatible con WooCommerce Subscriptions
* Fácil configuración y uso
* Seguridad garantizada en las transacciones

== Instalación ==
1. Descarga el plugin y súbelo a la carpeta `/wp-content/plugins/`.
2. Activa el plugin desde el menú "Plugins" en WordPress.
3. Configura el plugin en Woocommerce -> ajustes -> Pagos -> Suscripción con Wompi
4. Comienza a aceptar pagos por suscripción en tu tienda online.
